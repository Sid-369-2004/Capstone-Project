In this capstone course, you will apply various Data Science skills and techniques that you have learned as part of the previous courses. 

You will assume the role of a Data Scientist who has recently joined an AI-powered weather data analytics company and will be presented with a challenge that requires data collection, analysis, basic hypothesis testing, visualization, modeling, and dashboard creation to be performed on real-world datasets.

### Tasks to Undertake
* Collecting and understanding data from multiple sources
* Performing data wrangling and preparation with regular expressions and Tidyverse
* Performing exploratory data analysis with SQL and visualization using Tidyverse and ggplot2
* Modeling the data with linear regressions using Tidymodels
* Building an interactive dashboard using R Shiny

The project will culminate with a presentation of your data analysis report, including an executive summary for the various stakeholders in the organization. You will be assessed on both your work throughout the various stages of the data analysis process, as well as the final deliverable.

This project is a great opportunity to showcase your Data Science skills and demonstrate your proficiency to potential employers.

The final task of this capstone project is to create a presentation based on the outcomes of all tasks in previous modules. Your presentation will develop into a narrative of your analysis that should be compelling and easy to understand.

Your submission will be **auto-graded by AI** using a tool called **Mark**.
