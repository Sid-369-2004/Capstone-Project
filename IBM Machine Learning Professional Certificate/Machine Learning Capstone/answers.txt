Question 2: Choose the correct option based on the approach used in the content-based recommender system.
---------------------------------------------------------------------------------------------------------
Answer: Recommendations are generated by matching the user profile (built from the genres of courses the user previously interacted with) with the genre features of available courses.
Explanation: In content-based filtering using user profiles and course genres, a user profile vector is generated by multiplying the user's historical course interaction vector with the course-genre matrix. Recommendations are then generated by measuring the similarity (e.g. dot product or cosine similarity) between this user profile vector and the genre feature vectors of unselected courses.


Question 3: Select the correct options based on the steps that are part of the KNN-based recommender workflow.
-------------------------------------------------------------------------------------------------------------
Answer (Select all that apply):
  - Construct a user-item interaction/rating matrix.
  - Calculate similarity scores (e.g., Cosine similarity or Pearson correlation) between users or items.
  - Identify the K nearest neighbors based on the similarity scores.
  - Predict ratings or generate recommendations based on the ratings of the identified K nearest neighbors.
Explanation: The K-Nearest Neighbors (KNN) recommender workflow for collaborative filtering involves:
  1. Creating a pivot table representing the user-item interaction matrix.
  2. Computing a similarity matrix (using metrics like cosine similarity) to find the nearest users (user-user) or items (item-item).
  3. Selecting the top K neighbors with the highest similarity scores.
  4. Predict the target user's interaction/rating by aggregating the neighbors' ratings.


Question 4: Choose the correct option based on which metric was used to evaluate the collaborative filtering models?
----------------------------------------------------------------------------------------------------------------------
Answer: Root Mean Squared Error (RMSE)
Explanation: Collaborative filtering models (such as KNN, NMF, and Neural Network Embeddings) are evaluated and compared using the Root Mean Squared Error (RMSE) metric to assess the accuracy of predicted ratings compared to actual user ratings in the test set.
